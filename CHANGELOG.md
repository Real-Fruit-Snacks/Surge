# Changelog

All notable changes to Surge will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-09

### Added
- Markdown-to-searchable command reference transformation
- Fuzzy search with real-time results
- Variable substitution with custom values
- Catppuccin Mocha/Latte theme support
- Offline-first PWA with service worker
- Category and tag filtering
- Copy-to-clipboard with variable replacement
- Static site — no server required
