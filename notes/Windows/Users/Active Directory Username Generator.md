## Active Directory Username Generator
resources: [AD-Username-Generator GitHub](https://github.com/mohinparamasivam/AD-Username-Generator)
### Generate Usernames
```bash
python3 username-generate.py -u /root/machines/<Machine>/<TargetIP>/usernames -o /root/machines/<Machine>/<TargetIP>/generated_usernames
```
