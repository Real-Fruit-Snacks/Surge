## Client Fingerprinting

> Gather OS and browser info from targets on internal networks before launching attacks. Use tracking links.

### Canarytokens
> Free service that generates tracking links. When clicked, reveals IP, browser, OS info.

```
https://canarytokens.org/generate
```

### Create Tracking Token
> Select "Web bug / URL token" -> Enter webhook URL -> Click "Create my Canarytoken"

### Other Fingerprinting Methods
> [!info] Alternative fingerprinting tools:
> - **Grabify** - IP logger service
> - **fingerprint.js** - JavaScript fingerprinting library

### Canarytoken File Types
> [!tip] Can embed tokens in Word documents, PDFs, or images that trigger when opened.
