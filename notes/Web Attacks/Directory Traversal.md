### Windows Specific Files [optional]
> [!info] Useful target files on Windows systems:
> - **C:\Windows\System32\drivers\etc\hosts** - Always readable, good for testing
> - **C:\inetpub\logs\LogFiles\W3SVC1\\** - IIS log files
> - **C:\inetpub\wwwroot\web.config** - May contain passwords
