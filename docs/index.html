FEATURES

Markdown in, commands out

Write notes in Obsidian, view them as a searchable command reference. Fuzzy search, variable substitution, and full offline support.

Fuzzy Search

Real-time search across titles, categories, tags, notes, and commands powered by Fuse.js. Sub-50ms response times across the entire index with highlighted matches and relevance ranking.

Variable Substitution

Define <variables> in commands that get highlighted in the UI. Set values once and they auto-substitute into every command on copy.

Workspaces

Save named variable sets for different environments. Switch between Lab, Production, and custom workspaces without re-entering values.

Keyboard Navigation

Full keyboard-driven workflow. Search, navigate, expand, copy -- all without touching the mouse. Seven keybindings for power users.

Offline-First PWA

Full functionality without internet after first visit. Service worker caches all assets locally. Zero external requests at runtime -- everything is bundled into the static site. Works from any web server or opened locally.

Toggle visibility of procedures by custom tags. Filter state persists across sessions. Configure default visibility per tag in config.js.

ARCHITECTURE

Build once, search anywhere

Python build script parses markdown into JSON. Single-page app consumes the index. No runtime server required.

1

Build Script

build.py parses markdown files from notes/ into a JSON search index. Standard library only -- no pip dependencies. YAML frontmatter extraction with tag support.

2

Search Engine

Fuse.js fuzzy search with relevance ranking and highlighted matches. Sub-50ms across the entire index. Searches titles, categories, tags, notes, and commands simultaneously.

3

Variable Engine

<variable> detection and substitution with workspace support. Values stored in localStorage. Auto-substitute on copy with visual highlighting in the UI.

4

Static Shell

Single-page app (~1500 lines). Service worker caches everything for offline. Four Catppuccin themes via CSS custom properties. Zero external requests at runtime.

GETTING STARTED

Build and browse

Python 3.7+ and any modern browser. No npm, no bundler, no framework.

build.sh
$ git clone https://github.com/Real-Fruit-Snacks/Surge.git
$ cd Surge

# Build search index from markdown
$ python3 build.py

# Start local server
$ python3 -m http.server 8080 --directory site
Serving on http://localhost:8080
